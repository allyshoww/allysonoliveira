# allysonoliveira.com.br

Blog pessoal do Allyson Oliveira — Arquiteto de Soluções na AWS.

**Stack:** Hugo + tema Soho + Netlify

## Estrutura

```
content/
├── about.md          # Sobre mim (também é a home)
├── contact.md        # Contato
├── palestras.md      # Palestras e eventos
├── projetos.md       # Projetos
└── posts/            # Blog posts
```

## Desenvolvimento local

```bash
hugo server -D
```

## Deploy

Push para `master` → Netlify faz o build automaticamente com Hugo.
